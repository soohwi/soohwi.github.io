📌 프로젝트 개요

사용자 중심의 메모 관리 기능을 갖춘 메모장을 Vue3 기반으로 개발했습니다. SPA(Single Page Application) 구조로 설계하고, 사이드바와 에디터 영역으로 분리된 직관적인 레이아웃을 구성했으며, 페이지 추가/수정/삭제 및 다크모드, LocalStorage 저장 기능을 구현한 프로젝트입니다. 기존 포트폴리오 구조 내 모달 연동 및 Iframe 임베딩 방식을 적용하여 작업물을 효과적으로 연결했습니다.

🛠 주요 역할 및 기여
  • Vue 기반 SPA 설계 및 컴포넌트 개발
    • Sidebar, Editor, DarkModeStore 등 주요 UI 영역을 컴포넌트 기반으로 설계하여 재사용성과 유지보수성을 높였습니다.
    • Pinia를 활용한 전역 상태 관리로 페이지 간 데이터 공유를 효율적으로 처리했습니다.
  • 페이지 관리 기능 구현
    • 페이지 추가/선택/수정/삭제 기능을 구현하고, 사용자의 페이지 데이터가 LocalStorage에 자동 저장되도록 설정하여 UX를 강화했습니다.
  • 다크모드 전환 기능 개발
    • 사용자 환경에 맞춰 다크/라이트 모드를 전환할 수 있도록 전역 상태 관리 및 CSS 변수 시스템을 적용하였습니다.
    • :root 변수와 class 토글을 통해 글로벌 스타일을 유연하게 변경 가능하도록 구성했습니다.
  • 퍼블리셔 경험 기반의 UI/UX 최적화
    • 인터랙션 및 전환 효과, 마진/패딩, 사용자 편의성을 고려한 마크업 구조 설계로 UI 완성도를 높였습니다.
    • 불필요한 의존성 없이 Vue 내장 기능만으로 가볍고 직관적인 코드 구조를 구현했습니다.
