📌 프로젝트 개요

내셔널지오그래픽 브랜드의 온라인몰 ‘N.Station’을 리뉴얼하며, 웹/모바일 전반의 UI 개선과 퍼포먼스 향상을 목표로 프로젝트를 진행하였습니다.

🛠 주요 역할 및 기여
  • UI 최적화 및 컴포넌트 모듈화: 웹/모바일/어드민의 메인·서브 레이아웃을 설계하고, 공통 UI 컴포넌트를 개발하여 재사용성과 일관성 확보
  • 생산성 향상을 위한 가이드 정립: 업무 가이드 및 코드 가이드라인을 문서화하여 협업 효율성 및 신규 인력 온보딩 지원
  • JavaScript 기반 인터랙션 구현: 인기 검색어 롤링 모션을 라이브러리 없이 직접 구현하여 사용자 경험 강화
🧩 전체 레이아웃 & UI 구조
  • LOGIN, JOIN
    모바일 로그인 & 회원가입 페이지는 팝업 형태의 레이아웃을 사용하며 직접 작업하였습니다.
    1. 로그인,회원가입 팝업 레이아웃 작업
    2. Icon Font로 스마일 아이콘을 로드하여 CSS로 색상 변경
    3. 폼 입력 유형들과 버튼 및 안내문구를 공통 UI 컴포넌트로 작업
    4. 폼 입력 유형들과 버튼 및 안내문구를 공통 UI 컴포넌트로 작업
    5. 모달 팝업 레이아웃 작업
  • 주문결제, 장바구니

    Mobile 주문결제 & 장바구니 페이지에 사용되는 모든 컴포넌트를 직접 작업하였습니다.

    • 모든 영역은 드롭다운 형태로 열림/닫힘 기능을 갖춤. 배송지 정보 영역에서 ‘배송지를 등록하세요’를 클릭하면 모달 창이 열리고 배송지를 등록한 후에 라디오 버튼이 생성
    • 장바구니와 주문/결제 페이지에서 선택된 상품 리스트가 동일한 UI를 사용하므로, 확장성을 고려하여 공통 UI 컴포넌트로 개발
  • MYPAGE 쇼핑몰 내 고객의 공간 마이페이지
    • b로 기본 bar 형태 만든 후, 내부에 i 생성해 개발팀에서 수정할 수 있도록 inline-style로 title=“50%” style=“width: 50%;” 등 값 조절
    • ol구조이며, ol에 CSS flex 속성을 정의하고 li에 flex:1을 사용하여 디바이스 크기에 따라 유연하게 늘어나거나 줄어듦
    • MY 찜 페이지에서 ‘+ 새 폴더 추가하기’ 버튼 클릭 시 노출되는 Modal 레이아웃 작업
  • LOGIN, JOIN: PC 로그인 & 회원가입 페이지이며, 엔스테이션 메인 레이아웃 작업을 진행하였습니다.
    • SNS 아이콘 폰트는 클래스명에 따라 로드되는 아이콘과 배경색이 다르도록 SCSS @each 문법 사용
    • PC 모달 팝업 레이아웃 작업
    • 폼 입력 유형들과 버튼, 안내문구 등 공통 UI 컴포넌트 작업
  • 주문결제 & 장바구니: PC 주문결제 & 장바구니 페이지에 사용되는 모든 컴포넌트를 직접 작업하였습니다.
    • 여러 페이지에서 사용되는 상품 리스트를 공통 UI 컴포넌트로 작업 CSS flex 속성을 사용하여 내부 아이템들을 정렬하고 간격을 조정
    • 배송지 등록했을 경우 slider로 리스트 노출
    • 쿠폰 적용 버튼 클릭 시 노출되는 모달 레이아웃 작업
  • MYPAGE 쇼핑몰 내 고객의 공간 마이페이지
    • b로 기본 bar 형태 만든 후, 내부에 i 생성해 개발팀에서 수정할 수 있도록 inline-style로 title=“50%” style=“width: 50%;” 등 값 조절
    • MY 찜 페이지 레이아웃으로, 폴더 리스트는 height 값 고정 후 세로 스크롤 기능 추가
    • 마이페이지 전체 레이아웃에 nav, section 등 시맨틱 태그를 사용하고, 웹 표준, 웹 접근성을 준수하여 다양한 브라우저에 맞는 UI 개발 진행
