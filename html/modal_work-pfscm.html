📌 프로젝트 개요

판매, CS, 물류, 배송 업무 등 다양한 관리자 기능이 포함된 SCM 관리자 페이지를 구축하였습니다. 공통 레이아웃 및 구조 설계를 통해 유지보수성을 높였습니다.

🛠 주요 역할 및 기여
  • 관리자 메인 레이아웃 설계: HTML5, SCSS 기반으로 메인 페이지 UI를 설계하고, 직관적인 사용자 인터페이스 구성
  • 표준화 기반 UI 구축: 다양한 관리자 기능을 공통 UI 패턴으로 통합하여 약 120여 개 페이지를 일관되게 개발
  • 유지보수성 향상: 시맨틱 마크업을 활용하여 유지보수 편의성과 코드 일관성 확보
