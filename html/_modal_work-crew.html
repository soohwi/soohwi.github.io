📌 프로젝트 개요

온라인 패션 쇼핑몰 '스타일크루'의 웹/모바일/어드민 UI를 신규로 설계 및 개발하였으며, 다양한 디바이스에 최적화된 사용자 경험을 제공하는 데 초점을 맞춘 프로젝트입니다.
디자인 가이드 정립부터 Git 기반 협업까지 프로젝트 전체 퍼블리싱을 주도하였습니다.

🛠 주요 역할 및 기여
  • UI 레이아웃 개발: 웹/모바일 앱/어드민의 메인·서브 페이지 전체 레이아웃을 직접 설계 및 구현하여 디바이스별 일관된 사용자 경험 제공
  • 인터랙티브 UX 구현: CSS Animation과 JavaScript를 활용한 자연스러운 애니메이션 효과로 사용자 몰입도 향상
  • 공통 컴포넌트 설계: 확장성과 유지보수를 고려한 공통 UI 컴포넌트 모듈화 및 개발 가이드 정립
  • 협업 환경 구축: Git Flow 전략에 따른 브랜치 분기 및 협업 프로세스 수립으로 안정적인 배포와 협업 효율성 강화
