📌 프로젝트 개요

모바일 중심의 HR 플랫폼 고도화를 위해 Vue.js 기반 UI를 설계하고, Drag & Drop 기능 및 데이터 시각화를 통해 사용자 경험을 향상시킨 프로젝트입니다.
SCSS mixins 기반의 확장 가능한 스타일 구조와 프로토타이핑을 통한 빠른 UI 개선 프로세스를 주도하였습니다.

🛠 주요 역할 및 기여
  • Vue.js + JavaScript 기반 UI 설계 및 기능 구현: Drag & Drop 기능 및 대시보드 데이터 시각화(ApexCharts 활용)
  • SCSS mixins 기반 스타일 시스템 구성: 코드 재사용성 향상으로 생산성 30% 개선
  • JavaScript + CSS Animation 활용: 인터랙티브한 스크롤/이벤트 UX 구현
  • 모바일웹 아이콘 시스템 구축: SVG 기반 아이콘 폰트 시스템 적용
  • 빠른 프로토타이핑 기반 사용자 피드백 반영: 반복 개선을 통한 UX 완성도 향상
