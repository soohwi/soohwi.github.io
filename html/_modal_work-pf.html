📌 프로젝트 개요

대형 패션 커머스 플랫폼의 UI/UX 리뉴얼 프로젝트로, 반응형 웹 구축과 접근성 향상에 초점을 맞추어 다양한 사용자 환경에서 일관된 UI를 제공하였습니다.

🛠 주요 역할 및 기여
  • UI 리뉴얼 및 반응형 최적화: 다양한 디바이스 대응을 위한 반응형 웹 레이아웃 설계
  • 컴포넌트 설계: 재사용 가능한 공통 UI 컴포넌트 구축으로 유지보수 편의성 향상
  • 접근성 준수: WCAG 2.0 기준을 적용하여 시맨틱 마크업 및 ARIA 속성 도입
🧩 전체 레이아웃 & UI 구조
  • 상품 상세 페이지 mobile
    웹 표준과 웹 접근성을 준수하여 PC, Mobile 상품 상세 페이지의 전체 구조 및 세부 콘텐츠를 작업하였습니다.
    상품정보
    이미지 상단 ‘초특가 Deal’같은 이벤트 성 문구는 p 태그로 간결히 작업하고 이미지 하단 상품명 등 문구는 CSS ellipsis로 말줄임 추가
    상품리스트
    기존 공통 UI 컴포넌트로 만든 상품리스트에 sm 클래스를 추가하여 해당 디자인으로 작업
    기타정보
    dl, dt, dd 구조를 사용하고 세부 내용은 switch 기능을 활용하여 열림/닫힘 기능 작업
    탭메뉴
    상황에 따라 디자인만 추가할 수 있도록 탭메뉴의 구조만 공통 컴포넌트로 제작, 해당 탭메뉴는 flex:1을 사용하여 화면 비율에 따라 유연하게 늘어나고 줄어들 수 있도록 작업
  • 상품 상세 페이지 Web
    웹 표준과 웹 접근성을 준수하여 PC, Mobile 상품 상세 페이지의 전체 구조 및 세부 콘텐츠를 작업하였습니다.
    상품상세 left
    float: left로 나눈 영역으로, 상품 이미지 및 세트 추천 상품 영역은 Carousel 기능 사용
    상품상세 right
    float: right로 나눈 영역으로, 상품 정보에 관련된 모든 문구는 CSS ellipsis로 말줄임을 추가하고 switch 기능을 활용하여 열림/닫힘 기능 구현
    옵션선택
    드롭다운 기능을 활용하여 옵션 리스트를 노출시키고, 선택된 옵션은 옵션 선택 버튼 하단에 노출
