📌 프로젝트 개요

대형 카테고리 중심의 온라인 쇼핑몰을 신규로 구축하였으며, 슬라이드 메뉴 등 주요 UI 요소를 포함해 사용자 편의성을 높였습니다.
공통 UI 컴포넌트를 적극 활용하여 페이지를 구성하였고, 퍼블리셔 중심으로 기획-디자인-개발과의 협업을 진행하였습니다.

🛠 주요 역할 및 기여
  • 공통 UI 컴포넌트를 활용하여 서브 페이지 개발
  • 대형 카테고리, 슬라이드 메뉴 개발 및 최적화
